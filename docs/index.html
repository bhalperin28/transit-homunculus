Time-space cartogram

Streets redrawn so distance reflects travel time, not physical geography — for any city. Pick a generated city below, or generate your own with the pipeline.

No cities generated yet — run the pipeline below.

Generate a new city

npm install
npm run pipeline -- --city "Portland, Oregon"

Geocodes the city with OpenStreetMap Nominatim, pulls its road network and transit routes from Overpass, computes travel-time matrices for free-flow driving, rush-hour driving (a modeled congestion profile), and transit, then embeds each into 2D with stress-majorization MDS. Output lands in docs/<city-slug>/ and this page updates itself. See the README for details and options (--radius, --anchors).

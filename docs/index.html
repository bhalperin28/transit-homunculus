Prefer more control?

The search box above runs the whole pipeline — road network, transit routes, travel-time routing, and the 2D embedding — live in your browser via a Web Worker, using Nominatim and Overpass. For a larger anchor count, a custom radius, or to save the result as a permanent page in this repo, use the CLI instead: 

npm install
npm run pipeline -- --city "Portland, Oregon"

See the README for details.
